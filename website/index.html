OatmealUtils

An Open Source Library

  • What is it?

    This is a C# library of classes that we have found useful, and we are giving out to the broader community in the hope that you may also find them useful. We are initially distributing it to support the FastDAO and Geocoding projects, but some of the utilities are handy enough to use all by themselves. This library includes collection classes that throw more detailed exceptions than the standard collections, caches for reusing objects, a config file parser, string parsing utilities, and more. For full details, see the API Docs below.

  • Why Oatmeal?

    Oatmeal and utility libraries. They're both thought of as bland or boring, but can be the foundation for great things.
